How to use

  1. Prepare a CSV. First row is headers; each column name maps to a Figma layer named with the # prefix (e.g., #Title, #Image).
  2. In Figma, select the frame/group whose children contain layers named with # columns you want to populate.
  3. Click Upload CSV. The plugin will duplicate the selection to match row count and populate from the CSV.
  4. Optional: Enable “Rename with numbers” to auto-number when no matching header exists.
