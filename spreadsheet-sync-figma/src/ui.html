How to use

  1. Prepare a CSV/XLSX file or use a public Google Sheet. First row is headers; each column name maps to a Figma layer named with the # prefix (e.g., #Title, #Image).
  2. In Figma, select the frame/group whose children contain layers named with # columns you want to populate.
  3. Either upload a file or paste a Google Sheet URL, then click Sync. The plugin will duplicate the selection to match row count and populate from the data.
  4. Optional: Enable "Rename with numbers" to auto-number when no matching header exists.
Make sure your Google Sheet is set to "Anyone with the link can view"
